Install via pipÂ¶

pip is the package installer for Python. You can use pip to install packages with dependencies from the Python Package Index and other places.

  • Python Package Index (PyPI)

pip install sphinx

  • Local directory

pip install .

  • GitHub from (branch, release, tag, commit)

pip install git+git://github.com/author/repository.git@master

  • From the private packages

pip is just python script, when you run it with python you will do work for that python. If you run script without explicit python executable, you will use latest installed pip.

> pip.exe -V
pip 19.2.3 from h:\python27\lib\site-packages\pip (python 2.7)

You can run pip with explicit Python version

> c:\Python38\python.exe -m pip -V
c:\Python38\lib\site-packages\pip (python 3.8)
