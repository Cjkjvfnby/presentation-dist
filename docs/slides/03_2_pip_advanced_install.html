Working with packages¶

Specify index URL¶

If you want to install from another index you can specify index URL. Service should be compatible with PEP503.

Ways are listed in priority order, top item has more precedence.

  • pip install –index-url <url> ….

  • environment variable PIP_INDEX_URL

  • index-url option in pip.conf

Specify version to be installed¶

  • Exact version: ==

  • Lower bound: >, >=

  • Upper bound: <, <=

Get list of installed packages with their version¶

> pip freeze

mccabe==0.5.3

pyparsing==2.4.6

Requirements file for flake8¶

requirements.pip¶
pyflakes >= 2.1.0, < 2.2.0
pycodestyle >= 2.5.0, < 2.6.0
mccabe >= 0.6.0, < 0.7.0
