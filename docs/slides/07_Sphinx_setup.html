Setup Sphinx¶

Install Sphinx and style¶

requirements.pip¶
Sphinx==2.4.4
sphinx-rtd-theme==0.4.3

pip install -r requirements.pip

Run sphinx-quickstart and answer all hist question to create a project base.

Configure required extensions¶

conf.py¶
extensions = [
    'sphinx.ext.autodoc',  # autdoc module
    'sphinx.ext.intersphinx',  # links to external documentation services
    'sphinx.ext.viewcode',  # show code source in generated documentation
]

# Configure mappings for sphinx.ext.intersphinx
intersphinx_mapping = {'python': ('http://docs.python.org/3.8', None)}

Generate an HTML¶

make html
