Install package
***************

Ways to install a python package:

- download and do **python setup.py**
- download and copy to PYTHON_PATH
- use **easy_install**
- use **pip**

..
    Список основных способов представлен на слайде.

    Рассмотрим их подробнее.

    Использование сетап пай это низкоуровневый механизм установки питоновских библиотек.
    Но в тоже время это не самый удобный способ, так как требует нескольких действий: получить код, выполнить скрипт.
    В документации к проектом такой способ уже не указывается.

    Ручная установка библиотек обладает всеми недостатками предыдущего метода, да еще и приносит несколько своих.
    Необходимо вручную изготавливать скрипты и делать из исполняемыми.
    Но такой метод имеет место быть в случае если библиотека не предназначена для установки или окружение не позволяет использовать внешние библиотеки.
    Некоторые библиотеки намеренно пишутся в одном файле, чтобы их было удобно встраивать.  В качестве примеров можно привести six, pytest, bottle.

    easy_install он как интернет эксплорер, устарел и нужен только чтобы установить пип.
    Сейчас во всех установках питона с которыми я сталкивался пип поставляется с питоном из коробки, так что скоро он будет забыт.
